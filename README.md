# Salifort Motors Employee Attrition Analysis

## Project Overview

Salifort Motors aims to better understand the factors that lead employees to leave the organization. High employee turnover can increase operational costs, reduce productivity, and negatively impact organizational stability.

This project uses data analysis and machine learning techniques to identify patterns in employee data and predict potential attrition. The goal is to provide actionable insights that can help stakeholders develop strategies to improve employee retention.

---

## Business Problem

Salifort Motors wants to determine:

- What factors influence employee attrition
- Which employees are at higher risk of leaving
- What strategies can help reduce employee turnover

By analyzing historical employee data, this project builds predictive models that help the company proactively address retention issues.

## Project Objectives

- Analyze employee data to identify key factors influencing attrition

---

## Data Analysis Process

### 1. Data Exploration
- Inspect dataset structure
- Identify missing values and anomalies
- Analyze distributions and relationships between variables

### 2. Data Cleaning
- Handle missing data
- Remove duplicates
- Transform variables where necessary

### 3. Feature Analysis
- Identify important employee-related features such as:
  - Job role
  - Work hours
  - Salary
  - Job satisfaction
  - Work-life balance
  - Years at company

### 4. Model Selection
Several machine learning models are evaluated to determine the best performer, such as:

- Logistic Regression
- Decision Tree
- Random Forest
- Gradient Boosting Models

### 5. Model Evaluation
Models are evaluated using metrics such as:

- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix
